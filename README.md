# Sózdik — тренажёр казахской лексики

Учи казахские слова методом **интервального повторения** (spaced repetition,
как Anki/Duolingo). Двуязычный интерфейс KK/RU, прогресс без регистрации.

- **Наборы** по темам (быт, еда, путешествия, семья, числа) — 60 слов на старте
- **Карточки**: казахское слово → вспомни перевод → оцени себя (Снова/Трудно/Хорошо/Легко)
- **Алгоритм SM-2 lite** планирует следующее повторение каждого слова
- **Прогресс**: серия дней 🔥, повторено сегодня, изучено слов, по каждому набору

## Стек
- **Next.js 16** (App Router) + TypeScript + Tailwind v4
- **libSQL / Turso** — БД (локальный файл в dev, облачный Turso в проде)
- Прогресс привязан к анонимному устройству (cookie `did`) — без логина

## Локальный запуск
```bash
npm install
npm run dev        # http://localhost:3000
```
БД `sozdik.db` создаётся и наполняется автоматически при первом запросе.

## Деплой
Бесплатно, без карты: **Vercel + Turso** — см. [DEPLOY_VERCEL.md](DEPLOY_VERCEL.md).
С постоянной БД и автодеплоем на каждый пуш.

## Структура
```
app/            страницы + API (decks / review / progress)
lib/db.ts       БД (libSQL) — схема, сид, единственное место с SQL
lib/srs.ts      алгоритм интервального повторения (чистый, тестируемый)
lib/seed.ts     словарь (казахские слова с переводами)
middleware.ts   анонимный device-id
```
